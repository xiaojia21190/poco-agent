Preset 是一组可复用的 Agent 运行配置。通过预设，你可以为不同场景定义不同的 Agent 行为，并在项目和会话中快速应用。

## 可配置项

- **模型** — 指定默认使用的模型和供应商
- **能力开关** — 控制浏览器、记忆等能力的启用状态
- **工具** — 配置 MCP Server、Skills 和 Plugins
- **子 Agent** — 指定可调用的子 Agent 集合
- **图标与描述** — 为 Preset 设置视觉标识，方便识别

## 使用方式

- 在任务编辑器中选择 Preset，会话将按 Preset 配置运行
- 项目可以绑定默认 Preset，新建会话时自动继承
- 单次会话仍可覆盖 Preset 的任意配置项

## 管理入口

- 侧边栏 **Presets** 页面进行 CRUD 管理
- 任务编辑器中的 Preset 选择器支持快速切换
